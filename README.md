## 実行方法

### 依存関係のインストール

```bash
pnpm install
```

### ローカルサーバーの起動

```bash
pnpm run dev
```

### フォーマット

```bash
pnpm run format
```

### ビルド

```bash
pnpm run build
```

### ビルド結果のプレビュー

```bash
pnpm run preview
```
