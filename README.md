## 実行方法

### 依存関係のインストール

```bash
bun install
```

### ローカルサーバーの起動

```bash
bun run dev
```

### フォーマット

```bash
bun run format
```

### ビルド

```bash
bun run build
```

### ビルド結果のプレビュー

```bash
bun run preview
```
